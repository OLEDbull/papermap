# 【学习工作赛道】PaperMap · AI 科研知识图谱 - 初赛作品提交

## 作品介绍

PaperMap 是一个面向科研工作者的 AI 辅助文献调研工具。输入任意研究领域关键词，系统自动从多个学术数据库聚合论文，通过 AI 生成影响力评分、技术演进时间线、论文关系图谱和领域 Top10 核心论文，帮助研究者快速掌握领域全貌。

**核心功能：**
- 多源论文聚合（Semantic Scholar 期刊/顶会/综述 + arXiv 预印本）
- AI 批量论文分析（25篇/批，提速 10 倍）
- 技术演进时间线（自适应粒度 + AI 时段总结）
- 论文关系图谱（D3.js 力导向布局，聚类/自由双模式）
- 领域 Top 10 最具影响力论文（跨时间多样性选取）
- 多维分析报告（技术演进路线、核心研究者、入门路径、研究空白）

**技术栈：** Python Flask + DeepSeek API + Semantic Scholar API + arXiv API + D3.js + HTML5/CSS3/JS

## 项目亮点

1. **批量 AI 分析提速 10 倍**：DeepSeek API 调用从 200 次/搜索优化到 8 次/搜索
2. **多源数据智能聚合**：Semantic Scholar + arXiv 并行获取，智能去重
3. **真实引用量 + AI 估算双轨制**：优先使用真实数据，AI 估算补充
4. **技术演进全景可视化**：自适应时间粒度 + 横向滚动时间线 + 交互图谱

## 使用说明

1. 在顶部搜索框输入领域关键词（支持中文）
2. 等待系统自动检索、分析、生成图谱（约 1-2 分钟）
3. 横向滚动浏览技术演进时间线
4. 点击"关系图"切换视图，探索论文关联网络
5. 切换"分析"标签页查看深度分析报告
